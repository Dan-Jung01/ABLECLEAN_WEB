# ableclean-spa (React + MUI One-Page)

## Install
```bash
npm install
```

## Run
```bash
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Customize
- 전화번호: `src/sections/Contact.jsx` 의 표시 텍스트 및 `tel:` 링크 변경
- 지역/브랜드 문구: `src/components/layout/Navbar.jsx`, `src/sections/Hero.jsx`, `src/components/layout/Footer.jsx`
- 네이버 예약/카카오 채널: `src/sections/Contact.jsx` 버튼에 링크 연결
